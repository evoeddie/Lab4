Detailed Description

This class represents an incremental encoder connected to our given microcontoller.

Where we have the following attributes:

enc_name : Is the name of the encoder pin1 (pyb.Pin): This pin is representing encoder channel A.

pin2 (pyb.Pin): This pin is representing encoder channel B.

Timer (pyb.Timer): The timer used for the encoder counting. Using the manuel given for our board, we set timer 4 and timer 8 for our encoders.
